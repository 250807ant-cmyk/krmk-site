# КРМК — Казанский радиомеханический колледж

Лендинг колледжа: hero-видео, информация о колледже с раскрывающейся фотографией, направления (sticky-tabs в стиле Fixaplan), специальности (аккордеон), тест на профессию с фоновым видео, причины выбрать колледж, сообщество, scroll-stack CTA и футер с контактами.

## Стек
Чистый HTML + CSS + JavaScript. Никаких сборщиков, никаких зависимостей.

## Запуск
Просто открыть `index.html` в браузере, либо через локальный сервер:

```bash
python -m http.server 8000
# затем http://localhost:8000
```

## Структура
```
.
├── index.html      # вся разметка
├── style.css       # все стили
├── script.js       # все интерактивные блоки
└── assets/         # фото и видео
```
